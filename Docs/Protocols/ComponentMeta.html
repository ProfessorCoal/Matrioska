ComponentMeta

public protocol ComponentMeta

A protocol to create meta object that provides metadata for a Component

  • subscript(_:) Default implementation

    ComponentMeta should implement a subscript function to allow to retreive meta using a keyed subscript. A default implementation is provided and use reflection to retreive the values of the object’s properties.

    Parameter

    Parameter key: The key of the meta to retreive

    Default Implementation

    The default implementation of the subscript uses reflection to mirror the object if the key represent a property, its value will be returned

    Parameter

    Parameter key: the key of the value to retreive, must be the name of a property.

    Declaration

    Swift

    subscript(key: String) -> Any? { get }

    Parameters

    key

    The key of the meta to retreive
